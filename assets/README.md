# Assets

This directory should contain the following asset files for the app:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon for iOS and Android
   - Should be a square image with your app logo

2. **splash.png** (1284x2778 for iOS, 1080x1920 for Android)
   - Splash screen image
   - Should match the dark theme (#111217 background)

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Should work well on various backgrounds

4. **notification-icon.png** (48x48, optional)
   - Notification icon for Android
   - Simple, recognizable icon

## Creating Placeholder Assets

You can create simple placeholder assets using any image editor or use online tools:

1. For **icon.png** and **adaptive-icon.png**:
   - Create a 1024x1024 square image
   - Add your app logo or initials
   - Use gold (#D4AF37) as accent color

2. For **splash.png**:
   - Create a full-screen image
   - Use dark background (#111217)
   - Optionally add your app name in gold

3. For **notification-icon.png**:
   - Create a 48x48 simple icon
   - White icon on transparent background works well

## Temporary Workaround

If you don't have assets ready, Expo will use default placeholders. The app will still function normally, but you should replace them before publishing.
